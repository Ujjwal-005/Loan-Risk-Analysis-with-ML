# Loan Risk Analysis with Machine Learning

## Overview

This project predicts loan default risk using machine learning techniques and financial customer data.

The objective is to help financial institutions identify high-risk applicants and improve lending decisions through data-driven insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Streamlit
* Machine Learning

## Features

* Data preprocessing and cleaning
* Feature engineering
* Customer risk segmentation
* Loan default prediction
* Model evaluation and optimization
* Interactive Streamlit application

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Hyperparameter Tuning
6. Performance Evaluation
7. Deployment using Streamlit
